# 🌌 Star Wars Explorer

> Aplicación web educativa que consume SWAPI para explorar personajes, planetas y naves del universo Star Wars.

-----

## 📁 Estructura

```
starwars/
├── index.html
├── style.css
└── app.js
```

-----

## ✨ Funcionalidades

- 🧭 Navegación por tabs (Personajes / Planetas / Naves)
- 🔍 Buscador con filtrado local en tiempo real
- 🌍 Resolución de planeta natal con `Promise.all` (sin await en loop)
- 💾 Caché en memoria para evitar re-fetches innecesarios
- ⚠️ Manejo de errores con botón de reintento
- 📱 Diseño responsive inspirado en el universo Star Wars

-----

## 🌐 API utilizada

|API  |Endpoint                |Docs                            |
|-----|------------------------|--------------------------------|
|SWAPI|`https://swapi.info/api`|[swapi.info](https://swapi.info)|


> ⚠️ Se usa `swapi.info` en lugar de `swapi.dev` ya que este último se encuentra caído.

-----

## 🚀 Cómo correrlo

Abrí `index.html` directo en el navegador, o usá **Live Server** en VS Code.
